PROJECT NAME: Ui Framework presentation site

General Description: Сайт повинен складатись мінімум з трьох сторінок: реєстрації, логінізації та основної сторінки. Сайт розроблений для презентації власного UI Framework. Основне завдання - ознайомити користувачів з усіма можливостями вашого фреймворку.

Requirements:

  1. Сторінка реєстрації.
  2. Логін сторінка.
  3. Основна сторінка.

1. Поля для вводу паролю та емейлу повинні валідуватися. Після успішної реєстрації дані про користувача зберігаються в Local Storage.

2. Логінізація відбувається лише за умови наявності такого користувача в Local Storage.

3. На основній сторінці відображається весь функціоналвшого фреймворку, мінімальний набір: модальне вікно, панелі, головні контейнери, кнопки, таблиця.

Корстувач може копіювати код наведений у прикладі на вашому сайті.

Технології написання проекту необмежені.
